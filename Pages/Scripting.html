Cisco ICM Scripting



Los scripts de Cisco ICM se usan para que el sistema rutee cada llamada de acuerdo a una configuracion especifica

Por ejemplo teniendo en cuenta la hora de la llamada, dia, nro por el cual ingreso, numero del cliente que llama, si estan todos los agentes disponibles, etc


Tambien existen los scripts administrativos, que no rutean llamadas pero son usados para funciones especificas por los admin de la plataforma

Por ejemplo se podria usar para cargar una tablas de dias feriados del año, y establecer una variable que sea feriado booleana, para asi proporcionar un ruteo especifico

Tambien son usados por el dialer

Los scripts administrativos son los unicos scripts cuyos boxes son de color marron


Los scripts permiten invocar funciones nativas de cisco icm sencillas (matematicas, aritmeticas) o invocar funciones customizadas o incluso app desarrolladas por CVP

Las app del CVP son aplicaciones complejas que pueden realizar consultas a bases de datos, SQL , paginas WEB, etc etc

Estas funciones , consultas a base de datos, podrian servir por ejemplo para indicarle al cliente el saldo de su cuenta bancaria





  • CUIC (Reporting)
  • Dialer (campañas outbound / salientes - Discador)
  • ECE (modulo chat and e-mail)
  • Finesse (logueo de agentes)
